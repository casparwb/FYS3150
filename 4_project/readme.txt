The source folder contains all the code used for this project. All code is written in Julia. 
In the folder you will find the following files:

- functions.jl:- this file contains the general functions used to solve the problems, including the Monte Carlo simulation and the 
                initialization program
                
- mainprogram.jl: - the specialized code used to solve the specific tasks is in this file
- julia_parallel.jl:- parallel implementation of task e
- unittests.jl:- unit tests of a selection of the functions written for this project

The other folders are self explanatory.
